# GreenPulse AI

AI-driven insights for renewable energy deployment, carbon transparency, and sustainable growth in India.

## Features

- **Real-Time Dashboard**: Monitor renewable energy projects and carbon savings
- **AI Predictions**: ML-powered forecasts for energy and carbon metrics
- **Interactive Charts**: Visualize trends with Recharts
- **Responsive Design**: Beautiful UI built with TailwindCSS
- **Modern Stack**: React 18, Framer Motion, Lucide Icons

## Getting Started

### Prerequisites

- Node.js 16+ and npm

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

The app will open at `http://localhost:3000`

### Build

```bash
npm run build
```

## Deployment

This project is ready to deploy to Netlify, Vercel, or any static hosting service.

## Technologies

- React 18
- TailwindCSS
- Framer Motion
- Recharts
- Lucide React Icons
